## [0.1.0] - 2026-03-13

### Added

Initial public release of **PrivySHA**.

Core features:

- Prompt sanitization
- PII masking
- Prompt AST parsing
- Token optimization
- Context injection
- Prompt compilation pipeline

Adapters:

- OpenAI adapter
- Ollama adapter
- HuggingFace adapter

Developer features:

- Pipeline trace debugging
- Modular stage architecture
- Python package distribution
- PyPI installation support

Testing:

- Sanitizer tests
- Optimizer tests
- Pipeline tests
